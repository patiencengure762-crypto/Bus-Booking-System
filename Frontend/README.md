# Bus Booking System

A full-stack web application that allows users to book bus seats online.

## Features
- Passenger details input
- Seat selection
- Route selection between Kenyan counties
- Fare calculation
- Ticket preview
- Backend booking storage using JSON

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
